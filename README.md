# Título do Projeto

Este projeto é uma aplicação web simples desenvolvida em Python usando o microframework Flask. A aplicação permite gerenciar uma lista de jogos, implementando operações básicas de CRUD.


## Funcionalidades

- Flask==3.0.3
- Python 3.11.8
- Werkzeug==3.0.3
- WTForms==3.1.2
- SQLAlchemy==2.0.30
- Flask-Bcrypt==1.0.1
